🏠 Expense Tracker Pro (Room Expense Split System)
<p align="center"> A smart <b>Room Expense Management Application</b> built using <b>Python</b> and <b>Tkinter</b>.<br> Designed to track <b>monthly expenses</b> and split costs among roommates efficiently. </p> <p align="center"> <img src="https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python"> <img src="https://img.shields.io/badge/GUI-Tkinter-green?style=for-the-badge"> <img src="https://img.shields.io/badge/Data-CSV-orange?style=for-the-badge"> <img src="https://img.shields.io/badge/Status-Completed-success?style=for-the-badge"> </p>

## 🚀 Features
- 💰 Expense Management <br>
Tracks multiple expense categories:
   - 🏠 Rent
   - 🍔 Food
   - ⚡ Electricity
   - 📶 WiFi
   - 🚿 Water

 👥 Roommate Splitting
 - Add multiple roommate names
- Automatically calculates:
   - Total expense
   - Per-person share
Displays individual breakdown

📅 Month-wise Tracking
- Save expenses by month (e.g., 2026-05)
- Maintain historical records

💾 Data Storage
- Stores all data in a structured CSV file
- Easy to access and portable

📊 Professional UI Table
- Clean Treeview table (like Excel)
- Scrollable (vertical + horizontal)
- Alternating row colors for readability

🔄 Reset & Reuse
 - One-click reset for new entries

## 📌 How It Works
<details> <summary>🎮 Click to expand</summary>
1. User enters: <br>
  - Month
  - Expenses (Rent, Food, Electricity, WiFi, Water) <br>
  - Roommate names <br>
2. System calculates: <br>
  - Electricity bill <br>
  - Total expense <br>
  - Per-person share <br>
3. Data is: <br>
   - Displayed in UI <br>
   - Saved to CSV file <br>
4. User can: <br>
   - View history in table format <br>
   - Track monthly expenses <br>
</details>
